# Image Crawler

A powerful, sleek, and dark-themed web image crawler built with Python. 
Extracts image links from any website and generates a professional PDF report.

**100% vibe coded**

## Features

*   **Dark Modern UI**: A clean, responsive interface designed for comfort.
*   **Deep Crawling**: Recursively crawls pages to find all images.
*   **Smart Filtering**: Automatically filters out icons, tracking pixels, and non-image files.
*   **PDF Reporting**: Generates a beautiful PDF report with clickable links to all images found.
*   **Threaded Performance**: Runs smoothly in the background without freezing the UI.

## Installation

1.  **Install Dependencies**: Double-click `Install_Dependencies.bat`.
    *   This will check for Python and install required libraries (`requests`, `beautifulsoup4`, `reportlab`).
2.  **Start the App**: Double-click `Start_App.bat`.

## Usage

1.  Enter the target **URL** (e.g., `https://example.com`).
2.  Set the **Max Pages** limit (default: 50).
3.  Name your **Outcome PDF** (default: `images.pdf`).
4.  Click **Start Crawling**.
5.  Sit back and enjoy the vibes while the crawler does the work.

## Requirements

*   Python 3.7+
*   Internet Connection
